# Bkper Web

Web tooling and SDKs for building applications on Bkper.

## Packages

This monorepo contains the following packages:

- **[@bkper/web-design](./packages/design)** - CSS design system (variables, tokens, themes)
- **[@bkper/web-auth](./packages/auth)** - Framework-agnostic authentication SDK
- **@bkper/web-components** - Lit web components (coming soon)
